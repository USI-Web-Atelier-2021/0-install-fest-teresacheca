Goal

Welcome to the Web Atelier.

The goal of the InstallFest is to setup the programming environment that you will be working on for the rest of the atelier.

This is the Linux version of the installation instructions.

Task 1. Setup Google Chrome

  1. Get and install Google Chrome (https://www.google.com/intl/en/chrome/browser/)

Task 2. Setup Visual Studio Code and the Live Share Extensions

  1. Get and install Visual Studio Code (https://code.visualstudio.com/). If you use snaps, you can get it from https://snapcraft.io/code.

  2. Plug in the Live Share Extension Pack (https://marketplace.visualstudio.com/items?itemName=MS-vsliveshare.vsliveshare-pack). You may need to restart VS Code to complete the installation.

  3. Add the Preview on Web Server extension.

Note: While you are welcome to use your favourite editor to develop your Web applications, we will be using VS Code live sharing during the Mastery Checks and in case you need remote help.

Task 3. Install cURL (curl)

This is the command line tool for transferring data with URLs.

Install it with the following:

sudo apt-get install curl

Do not install using snap or the next steps will fail.

Task 4. Install Node Version Manager (nvm)

The following steps will install Node Version Manager (https://github.com/creationix/nvm). This program allows you to manage multiple versions of node.js in your machine.

  1. Make sure you have a .bash_profile file before installing. Open a new Terminal shell and run the following command:

    touch ~/.bash_profile

    to create the necessary profile if it does not exist

  2. Install NVM by typing the following command in the Terminal (one line command):

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash

    Running the above command downloads a script and runs it. The script clones the nvm repository to ~/.nvm.

  3. Once the installation is complete, open a new shell and type:

    nvm --version

    You should see an output similar to:

    0.38.0

  4. If you do not see the line above, then run the following command:

    bash

    To make sure you have re-initialized the shell and try again.

Task 5. Install node.js v15 (or later)

  1. Install node.js by typing the following command in the Terminal:

    nvm install 16

  2. Once the installation is complete, open a new Terminal and type:

    node --version

    You should see an output similar to:

    v16.9.1

Task 6. Install Yarn

  1. Open a Terminal window and type the following commands:

    npm install --global yarn

  2. Once the installation is complete, type:

    yarn --version

    You should see an output similar to:

    1.22.11

Task 7. Install nodemon

  1. Time to install nodemon, issue the following command:

    npm install --global nodemon

  2. Once the installation is complete, type:

    nodemon -v

    You should see an output similar to:

    2.0.12

Task 8. Install mongoDB

  1. We will use the official community edition.

  2. First Configure Ubuntu to trust the mongodb server

    sudo apt-get install gnupg
wget -qO - https://www.mongodb.org/static/pgp/server-5.0.asc | sudo apt-key add -

  3. Create the correct list file depending on your ubuntu version:

    • Ubuntu 20
      echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/5.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-5.0.list
    • Ubuntu 18
      echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu bionic/mongodb-org/5.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-5.0.list
    • Ubuntu 16
      echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu xenial/mongodb-org/5.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-5.0.list

  4. Update the local package list

    sudo apt-get update

  5. Finally install the latest version of mongodb

    sudo apt-get install mongodb-org

  6. To check if the mongoDB client is installed type:

    mongo --version

    You should see something like:

    mongoDB shell version: v5.0.2

Task 9. Install MongoDB Compass

  1. Download the .deb package for Ubuntu from the MongoDB Compass download page.
  2. Install the package by opening it and clicking on install.

Task 10. Install Postman

  1. Install it from https://snapcraft.io/postman or using the shell command line:
sudo snap install postman
  1. After opening it, create a free account or skip to just use the sandbox.
  2. It is not needed to set up a team, you can skip this step.

Task 11. Sign up and test your GitHub Classroom account

  1. You will need your GitHub account

  2. Follow this invitation link to work on Assignment 0

  3. Clone the repository

  4. Enter your name in the README.md

  5. Save the file

  6. Commit and push your change

    git add README.md
git commit -m "completed installfest"
git push

  7. Congratulations! you have completed the setup for the Web Atelier.
